# PROJECT_CONTEXT

## Vision
Refill List is a proactive household inventory and refill intelligence platform.

The product should:
- track consumables
- estimate depletion
- predict refill timing
- reduce forgotten purchases
- create intelligent refill suggestions

---

## Core Problem
Most shopping apps only manage lists.
Users still react after products are already empty.

The real problem:
- missing household state awareness
- no consumption intelligence
- no prediction system
- no adaptive refill behavior

---

## Strategic Direction
The application should evolve from:

shopping list
→ household inventory
→ consumption modeling
→ predictive refill system
→ autonomous household assistant

---

## Core Entities

- Household
- User
- HouseholdItem
- ConsumptionEvent
- RefillPrediction
- ShoppingSession

---

## Future AI Features

- Consumption forecasting
- Household behavior modeling
- Smart refill timing
- Recommendation systems
- Voice input
- OCR receipt scanning
- Barcode scanning

---

## Product Identity
The system should feel like:

"A system that understands the state of my household."

Not:

"A simple grocery checklist app."
